Weighted Correlation Analysis (WCA):

The Weighted Correlation Score calculates the relationship between stocks while incorporating predefined weights () for each stock: \[ WCA = {w^T}{C_w} \] w:Weight vector , C:Correlation matrix of stock return.

Show the code 
# Define weights based on arbitrary criteria (e.g., market cap or equal weights)

# Correlation matrix

weighted_correlation_scores = weights @ correlation_matrix @ weights.T

Portfolio Optimization

Portfolio optimization aims to allocate asset weights w to maximize the Sharpe Ratio. Using predicted returns and a covariance matrix (Σ), optimize weights (w) to maximize the Sharpe Ratio. \[ \text{Sharpe Ratio}= \frac{w^T \mu}{\sqrt{w^T \Sigma w}} \] Where, \[ \mu: \text{Expected returns vector, and derived using mean historical returns } \] \[ \Sigma: \text{Covariance matrix of stock returns, and derived using the covariance of returns} \] \[ w:\text{Portfolio weights (sum to 1)} \]

Optimization is performed using genetic algorithms (GA). -Solve for 𝑤 using PyPortfolioOpt to maximize the Sharpe Ratio -Portfolio_performance measure expected return, volatility, and Sharpe ratio

Show the code 
# Portfolio performance
from pypfopt import EfficientFrontier, risk_models, expected_returns

# Calculate expected returns and risk model
mu = expected_returns.mean_historical_return(data)
S = risk_models.sample_cov(data)

# Perform mean-variance optimization
ef = EfficientFrontier(mu, S)
weights = ef.max_sharpe()
cleaned_weights = ef.clean_weights()

# Portfolio performance
performance = ef.portfolio_performance(verbose=True)
print("Optimized Weights:", cleaned_weights)

Additional Financial Analysis

Incorporate advanced metrics like Value at Risk (VaR) and Conditional VaR (CVaR). 1. Value at Risk (VaR): Measures potential loss over a given time frame with a confidence level (𝛼): \[ VaR_{\alpha} = Percentile_{(1-\alpha) \cdot 100}(R_{t}^{Portfolio}) \] 2. Conditional Value at Risk (CVaR): Average loss beyond the VaR threshold: \[ CVaR_{\alpha} = E[R_{t}^{Portfolio} | R_{t}^{Portfolio} \leq VaR_{\alpha}] \]

Show the code 
#  Financial Analysis
# Value at Risk (VaR)
def calculate_var(returns, confidence_level=0.95):
      return np.percentile(returns, (1 - confidence_level) * 100)
# Conditional Value at Risk (CVaR)
def calculate_cvar(returns, confidence_level=0.95):
      var = calculate_var(returns, confidence_level)
      return returns[returns <= var].mean()

var = calculate_var(portfolio_returns, 0.95)

# Risk Parity Portfolio
import numpy as np
import pandas as pd
from pypfopt import risk_models, expected_returns
from scipy.optimize import minimize

# Function to calculate risk contributions
def calculate_risk_contribution(weights, cov_matrix):
    portfolio_variance = np.dot(weights.T, np.dot(cov_matrix, weights))
    marginal_contribution = np.dot(cov_matrix, weights)
    risk_contributions = marginal_contribution * weights / portfolio_variance
    return risk_contributions

# Objective: Minimize the difference in risk contributions
def risk_parity_objective(weights, cov_matrix):
    risk_contributions = calculate_risk_contribution(weights, cov_matrix)
    return np.sum((risk_contributions - np.mean(risk_contributions))**2)

# Risk parity optimization
def optimize_risk_parity(cov_matrix):
    num_assets = cov_matrix.shape[0]
    init_guess = np.ones(num_assets) / num_assets
    bounds = [(0, 1) for _ in range(num_assets)]
    constraints = {'type': 'eq', 'fun': lambda weights: np.sum(weights) - 1}
    
    result = minimize(risk_parity_objective, init_guess, args=(cov_matrix,), bounds=bounds, constraints=constraints)
    return result.x

# Calculate covariance matrix
cov_matrix = risk_models.sample_cov(data)

# Get risk parity weights
risk_parity_weights = optimize_risk_parity(cov_matrix)
